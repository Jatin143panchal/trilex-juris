# Tri Lex & Juris - Legal Website

A modern, responsive legal website built with React, TypeScript, and Vite.

## Features

- **Legal Chatbot**: AI-powered legal assistant for consultations
- **WhatsApp Integration**: Direct WhatsApp contact functionality
- **SEO Optimized**: Complete SEO optimization with meta tags and structured data
- **Responsive Design**: Mobile-first design with Tailwind CSS
- **Modern UI**: Beautiful gradient design with shadcn/ui components
- **Production Ready**: Vercel deployment configuration included

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn/ui
- **Icons**: Lucide React
- **Deployment**: Vercel

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start development server:
   ```bash
   npm run dev
   ```

## Deployment

This project is configured for Vercel deployment with custom domain support.

### Environment Variables

- `NODE_ENV`: production (auto-configured)

## Project Structure

```
src/
  components/          # React components
    ui/               # shadcn/ui components
    LegalChatbot.tsx  # AI chatbot component
    WhatsAppButton.tsx # WhatsApp integration
  pages/              # Page components
  hooks/              # Custom hooks
  lib/                # Utilities
```
